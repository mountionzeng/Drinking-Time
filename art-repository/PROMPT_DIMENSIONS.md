# 美术参考库提示词维度表格

## 概览
提示词通过**8个维度表**来工程化管理，确保每个艺术风格的指导都是结构化、可重现的。

---

## 维度表1：美术流派（最高优先级）

| 流派 | 艺术家风格 | 特征 |
|------|-----------|------|
| 水彩插画 | 日式水彩 | 柔和、宁静、清新 |
| 油画 | 印象派 | 温暖、厚重、怀旧 |
| 素描 | 写实素描 | 精细、对比强、立体感 |
| 数字插画 | 概念艺术 | 锐利、科幻、动感 |
| 扁平插画 | 现代风格 | 简洁、亲切、矢量 |

---

## 提示词生成流程

```
用户描述
    ↓
特征提取 → 维度匹配
    ↓
多维评分（美术流派 50% → 色调 30% → 情感 15% → 材质 5%）
    ↓
选取最佳参考
    ↓
按优先级组织维度表：
   【美术流派】→ 【色调】→ 【光线】→ 【情感】→ 【构图】→ 【材质】→ 【质感】
    ↓
最终提示词输入生成引擎
```

## 实际提示词示例

### 水彩示例
```
【美术流派】水彩插画（日式水彩风格）
【色调】cool-light 主要颜色：蓝色、绿色、白色
【光线】散射光，柔和均匀 时间：清晨
【情感】宁静、清新、梦幻
【构图】远景构图，天空占比大 镜头角度：平视
【材质】纸质、水彩笔触
【质感】柔和
```

### 油画示例
```
【美术流派】油画（印象派风格）
【色调】warm-light 主要颜色：金色、红色、褐色
【光线】侧光，主光源明确 时间：黄金时段
【情感】温暖、怀旧、抒情
【构图】人物半身，背景虚化 镜头角度：略微俯视
【材质】布料、油彩厚涂
【质感】厚重
```

### 素描示例
```
【美术流派】素描（写实素描）
【色调】neutral-dark 主要颜色：黑色、灰色、白色
【光线】明暗对比强烈，侧光
【情感】庄严、细致、立体感强
【构图】人物肖像，正面构图 镜头角度：正面
【材质】纸质、铅笔
【质感】精细
```

## 维度权重优化指南

当用户描述时，系统按如下优先级匹配维度：

1. **美术流派（50%）** - 最重要，决定整体风格
2. **色调（30%）** - 第二重要，影响视觉温度
3. **情感（15%）** - 第三重要，增加语义指导
4. **其他维度（5%）** - 材质、光线、构图等细节

这个权重分配确保即使用户描述模糊，系统也能首先保证美术风格的准确性。

---

## 扩展说明

- 每个维度表都可以动态扩展新的流派和搭配
- 系统支持维度间的交叉组合（如"冷色调的油画"）
- 缓存的特征向量通过这些表自动映射到提示词维度
- 用户反馈会更新维度表的权重和优先级
