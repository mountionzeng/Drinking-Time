# Drinking Time · 用户指南

> 把碎片灵感，倒进来，慢慢说成一部能交付的影像。

## 这是什么

Drinking Time 是一个面向影视、广告、短片视觉开发的创作工作台。它做两件事，连成一条线：

1. **理解**——把你手里乱的东西（参考图、参考片、剧本片段、分镜、brief、随手笔记，或者只是一段聊天）整理成结构化、可复用、能喂给模型的影视语言。
2. **创作**——基于这份结构去生成镜头、画面、视频片段，并允许你继续手工修、反复比、最后交付。

名字说的是一种创作状态：从灵感摄入到影像产出的那段时间。轻松进去，把天马行空翻译成专业影像，最后变成能交付的东西。

## 这是给谁的

需要把"模糊的感觉"变成"可执行的镜头"的人：影视/广告/短片的视觉开发、概念设计、分镜与导演方向工作者。尤其适合——你脑子里有画面，但不想被工具的表单和流程绑住。

## 它替你解决三个真问题

- **整理的苦**：参考素材一堆，没人想先建文件夹、命名、排序。这里——拖进来就行，系统先替你整理成时间轴和结构。
- **从想法到镜头的断层**：聊出来的灵感、写下的剧本，落不到"这一镜具体怎么拍"。这里把它一路推到镜头表。
- **改一点却被全量重做**：通用 AI 工具你只想改一句，结果整段重生成，原来对的部分也毁了。这里的原则是——**改一点，只改这一点；你改过的，就是你的**。

## 两条路，最后都汇到镜头表

你可以两条都用，也可以只走一条。

**素材路**——手上已有参考：
拖入图/片/剧本/brief/笔记 → 系统自动识别类型、按天排时间轴、提炼维度（情绪/光影/空间/镜头语言）→ 生成可复用的**环境模板** → 解构成**镜头表**。

**故事路**——只有一个模糊的感觉：
跟「小酌」聊一件小事 → 聊出**故事卡片**（一段段素材）→ 合成**剧本**（logline / 主题 / 情感弧线 / 分场）→ 落成**镜头表**（每镜的主体 / 动作 / 台词 / 机位 / 光影…）。

两条路的终点都是同一张镜头表——你真正要交付的生产清单。

## 核心体验

- **一切都能改**：故事卡片正文、剧本（标题 / logline / 每场画面与情绪 / 情感弧线）、镜头表（主体 / 动作 / 台词）——点进去直接改，失焦或回车即存，刷新还在。
- **选中一段，让 Agent 只改这段**：选中任意一段文字 → 旁边浮出输入框 → 写一句指令（"改克制点""把这句改成环境描写"）→ Agent 只重写这一段 → 给你"原文 vs 新版"对比 → 你点采纳才替换。不动周围任何东西。
- **你改过的就是你的**：碰过的镜头/卡片归你，后续 Agent 重生成不会把它默默冲掉（这是产品承诺的方向，正在逐步覆盖到每个面）。

## 快速上手

1. 打开 `/analysis`。
2. 选一条路：拖素材进「DROP ZONE」，或点"聊一个故事"跟小酌开聊。
3. 让系统把它整理出卡片 / 剧本 / 镜头表。
4. 不满意的地方——直接点着改；想让 AI 帮忙的——选中那段，给一句指令，看对比，采纳。
5. 镜头表就是你的交付清单，越改越像你自己的东西。

## 两条使用原则

- **你只做修正，不做搬运**：分类、排序、初稿，系统先做；你只在必要时改。
- **AI 先给草稿，你来定稿**：第一次生成是一个很标准的东西；改着改着，它才变成只属于你的那一版。这正是这个产品想要的样子。

## 名字的意思

Drinking Time——倒一杯，把桌上讲过的八卦、深夜没说完的话，慢慢说成好故事。它既是情绪，也是方法：让你轻松进入创作，把独特想象力，变成工业级、可交付的影像。
