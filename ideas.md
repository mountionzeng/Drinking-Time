# Drinking Time — Analysis Engine 设计头脑风暴

## 项目核心概念

Drinking Time 是一个影视视觉开发平台。Analysis Engine 是其分析模块，负责将碎片化参考素材结构化为影视环境模板和场景/镜头产出表。

**独特机制**：UI 主题色根据当天的**纳音五行**动态切换，每种五行对应一种饮品色系：

| 五行 | 饮品 | 色调方向 |
|------|------|----------|
| 金 | 啤酒 | 琥珀金、麦芽黄 |
| 木 | 龙井 | 嫩绿、翠绿 |
| 水 | 椰汁 | 乳白、椰壳棕 |
| 火 | 大红袍 | 深红、琥珀红 |
| 土 | 咖啡 | 深棕、焦糖 |
| 默认/夜间 | 可乐 | 焦糖深色、气泡感 |

## 方案 A：「液态工坊」— Liquid Foundry

<text>

### Design Movement
**工业液态主义 (Industrial Liquidity)**：将影视后期制作车间的冷硬感与饮品的流动质感融合。界面像一个被液体浸润的金属工作台。

### Core Principles
1. **流体边界**：所有面板边缘不是硬直线，而是带有微妙的液态波纹效果，像液体在容器壁上形成的弯月面
2. **金属底色**：深色背景带有拉丝金属质感，五行饮品色作为液态高光覆盖在金属表面
3. **工业层级**：信息层级通过"液面深度"来表达——越重要的信息越"浮"在表面
4. **呼吸感**：整体界面有缓慢的"液面起伏"微动画，暗示当天五行的能量流动

### Color Philosophy
基底是深灰拉丝金属色（#1a1a1e），五行饮品色作为"液态层"叠加：
- 啤酒/金：琥珀色液面光泽 (#d4a017 → #f5c542)，像阳光穿过啤酒杯
- 龙井/木：翠绿透光感 (#4a7c59 → #8fbc8f)，像茶汤在白瓷杯中的色泽
- 椰汁/水：温润乳白 (#f0ead6 → #faf3e0)，带一点椰壳的暖棕
- 大红袍/火：深琥珀红 (#8b2500 → #c0392b)，像浓茶在紫砂壶中的色泽
- 咖啡/土：焦糖深棕 (#3e2723 → #795548)，像意式浓缩的油脂层
- 可乐/默认：焦糖黑 (#1a0f0a → #3d1c02)，带气泡粒子效果

### Layout Paradigm
**沉浸式双层面板**：上半区是三栏横向流动布局（Drop Zone → Timeline → Template Draft），下半区是全宽数据表格。面板之间用"液面分隔线"（一条带波纹动画的细线）隔开，而非传统的直线或间距。

### Signature Elements
1. **液面波纹分隔线**：面板之间的分隔线是动态的液面波纹，颜色跟随当天五行
2. **饮品气泡粒子**：背景中有极其微妙的气泡/蒸汽粒子缓慢上升，不同饮品有不同粒子形态（啤酒=大气泡，龙井=蒸汽，可乐=小气泡）
3. **金属铭牌标签**：状态标签像嵌在金属面板上的铭牌，带有微妙的凹陷/浮雕效果

### Interaction Philosophy
交互像"液体倾倒"：拖拽素材时有液态流入动画，状态变化时有色彩渗透过渡，hover 时面板表面泛起涟漪。

### Animation
- 页面加载：液面从底部缓慢上升，填满工作区
- 拖拽导入：素材像落入液面一样产生涟漪
- 状态切换：颜色像墨水在水中扩散一样渐变
- 五行切换（跨天）：整个界面的液态层缓慢变色，像调酒师在调配新饮品

### Typography System
- 标题：**Playfair Display**（衬线，工业优雅感）
- 正文英文：**DM Sans**（几何无衬线，清晰专业）
- 中文：**PingFang SC**（系统字体，保证渲染质量）
- 数据/代码：**JetBrains Mono**（等宽，技术感）

## 方案 B：「茶室暗房」— Darkroom Teahouse

<text>

### Design Movement
**东方暗房美学 (Oriental Darkroom)**：将传统暗房的红光安全灯氛围与东方茶室的留白哲学结合。界面像一个在暗房中品茶的影视人的工作空间。

### Core Principles
1. **暗室底色**：主背景是接近纯黑但带有微妙暖色倾向的深色（不是冷灰，而是像暗房中被安全灯微微照亮的黑）
2. **光源即五行**：当天的五行饮品色不是作为"主题色"平铺，而是作为"光源"照亮界面——像暗房中唯一的光源决定了整个空间的色调
3. **留白即呼吸**：大量使用负空间，面板之间的间距比传统工具更大，像茶室中物件之间的距离
4. **墨迹层级**：信息层级通过"墨色浓淡"来表达，最重要的信息最浓（最亮），次要信息像淡墨

### Color Philosophy
基底是暗房黑（#0d0d0f 带 2% 暖色偏移），五行饮品色作为"光源"：
- 啤酒/金：温暖的琥珀光 (#e8a838)，像黄昏透过啤酒瓶的光
- 龙井/木：清冷的翠光 (#5d9b6a)，像阳光穿过茶叶的透射光
- 椰汁/水：柔和的月白光 (#e8dcc8)，像椰汁表面反射的柔光
- 大红袍/火：暗房安全灯红 (#a63d2f)，最契合暗房概念的色彩
- 咖啡/土：焦糖暖光 (#6d4c2e)，像咖啡馆角落的台灯
- 可乐/默认：极深焦糖 (#140a04)，几乎不可见的底光

### Layout Paradigm
**卷轴式纵向展开**：上半区不是传统三栏并排，而是采用"画卷展开"的方式——左侧 Drop Zone 固定，中间 Timeline 和右侧 Template Draft 可以像卷轴一样横向滚动展开。下半区的表格也是全宽但带有"宣纸"质感的底纹。

### Signature Elements
1. **光晕指示器**：当前五行的饮品图标在页面顶部，周围有对应颜色的柔和光晕，像暗房中的安全灯
2. **墨迹进度条**：readiness 进度条不是传统填充条，而是像毛笔墨迹渐渐浸润宣纸的效果
3. **印章式状态标签**：状态标签（idea_pool、production_ready 等）设计成印章/篆刻风格，但保持现代感

### Interaction Philosophy
交互像"研墨"：操作是缓慢、精确、有仪式感的。拖拽时有墨迹轨迹，点击时有印章盖下的微妙反馈，hover 时有灯光照亮的效果。

### Animation
- 页面加载：从中心向外扩散的光晕，像暗房灯被打开
- 拖拽导入：素材沿墨迹轨迹滑入时间轴
- 状态切换：像印章盖下，先有压力感再显现颜色
- 五行切换：光源颜色缓慢渐变，整个空间的色调跟随改变，像暗房换了安全灯滤片

### Typography System
- 标题：**Noto Serif SC**（宋体衬线，东方文化感）
- 正文英文：**Source Sans 3**（人文无衬线，温和专业）
- 中文：**PingFang SC**
- 数据/代码：**IBM Plex Mono**（等宽，带有一点人文气息）

## 方案 C：「调色台」— The Grading Desk

<text>

### Design Movement
**影视调色台美学 (Color Grading Console)**：直接借鉴 DaVinci Resolve、Baselight 等专业调色软件的界面语言，但注入饮品色彩的温度感。界面像一个真正的影视后期调色工作站。

### Core Principles
1. **专业工具感**：界面密度适中，信息紧凑但不拥挤，像专业调色软件的面板布局
2. **色彩即语言**：五行饮品色不只是装饰，而是作为"调色参考"贯穿界面——顶部有一条当天饮品色的色带，像调色软件中的参考色条
3. **面板可控**：每个面板都有清晰的边框和标题栏，可以理解为调色台上的不同监视器
4. **数据可视化优先**：readiness、deadline、priority 等数据用专业的可视化方式呈现，而非简单的文字标签

### Color Philosophy
基底是调色室标准深灰（#1e1e22，接近 DaVinci Resolve 的背景色），五行饮品色作为"调色参考色带"：
- 啤酒/金：麦芽琥珀 (#c89b3c → #f0c75e)，warm highlight
- 龙井/木：茶绿 (#4a7a5b → #7db88a)，midtone green
- 椰汁/水：椰乳白 (#d9cdb8 → #f2e8d5)，soft highlight
- 大红袍/火：岩茶红 (#923a2f → #c45a4a)，warm shadow
- 咖啡/土：浓缩棕 (#4a3228 → #7a5c4f)，rich midtone
- 可乐/默认：焦糖黑 (#120c08 → #2a1a10)，deep shadow

### Layout Paradigm
**监视器矩阵布局**：上半区三个面板像三台并排的监视器（Drop Zone / Timeline / Template Draft），每个都有独立的标题栏和状态指示灯。下半区是全宽的数据表格，像调色台下方的节点编辑器区域。顶部有一条窄色带显示当天的五行饮品色和名称。

### Signature Elements
1. **五行色带**：页面顶部一条 4px 高的色带，颜色跟随当天五行，旁边显示饮品名和五行属性
2. **监视器边框**：每个面板有 1px 的微妙边框，左上角有一个小圆点状态灯（绿=活跃，灰=空闲，红=异常）
3. **示波器式进度条**：readiness 进度条设计成类似音频/视频示波器的样式，带有刻度线

### Interaction Philosophy
交互像"操作调色台"：精确、高效、无多余动画。Hover 时面板边框微微亮起（像监视器被选中），拖拽时有精确的对齐辅助线，操作反馈即时且克制。

### Animation
- 页面加载：面板像监视器逐个"开机"，从黑屏到显示内容
- 拖拽导入：素材以缩略图形式精确落入时间轴槽位，有轻微的"吸附"动画
- 状态切换：颜色变化快速且精确，像调色软件中切换 LUT
- 五行切换：顶部色带平滑过渡，面板的强调色同步更新，整体过渡时间 0.8s

### Typography System
- 标题：**Space Grotesk**（几何无衬线，现代技术感但不冰冷）
- 正文英文：**Inter**（高可读性，适合密集数据界面）→ 但使用 Variable 版本，利用 optical sizing
- 中文：**PingFang SC**
- 数据/代码：**JetBrains Mono**（等宽，专业感）

</text>

## 选择结论

**选择方案 C「调色台 / The Grading Desk」**

理由：
1. 最契合 Drinking Time 的影视专业定位——用户是影视从业者，调色台界面语言对他们来说是最熟悉、最有信任感的
2. 五行饮品色作为"调色参考色带"的概念最自然，不需要额外解释
3. 信息密度和专业感最适合 Analysis Engine 这种生产工具
4. 动画克制但精确，不会干扰工作流
5. 布局最接近 UI Spec 文档中描述的三栏 + 宽表结构
